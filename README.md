# saludo_terminal
archivo de saludo para incluir en shell

> Incluirlo al final del archivo ~/.bashrc รณ ~/.zshrc

~~~
#Mensaje de Saludo
~/saludo.sh
~~~

## Importante: Instalar previamente cowsays

### Debian, Ubuntu
~~~
sudo apt update
sudo apt -y install cowsay
~~~

### Fedora
~~~
sudo dnf install cowsay
~~~

### Manjaro, Arch
~~~
sudo pacman -S cowsay
~~~
